# MovieWebApp
This project will be at the end a web application in which you can add watched movies,  
you will also be able to rate these movies and commpare your
rating with ratings from thousands of users.

# Technologies
FrontEnd:  
- React  
- HTML  
- CSS  
  
BackEnd:
- Node.js / Express.js
